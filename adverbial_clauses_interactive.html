九種關係，九種連接

Nine ways two ideas can hold each other

連接詞不是要背的，是要看出來的。
兩個句子放在一起，它們之間的「關係」決定了該用哪個連接詞。
先看出邏輯，再選語言形式——這是英文真正能用出來的順序。

並列 · 平衡
Parallel — two ideas, equal weight
=
and · while · as
— Example —
She was eating dinner while he was reading.
她一邊吃飯，他一邊看書。

遞進 · 加深
Progressive — the second goes further
not only ... but also · moreover
— Example —
Not only is he smart, but he is also hardworking.
他不只聰明，而且勤勞。

條件 · 觸發
Condition — if this, then that
+ =
if · as long as · unless
— Example —
As long as you keep practicing, you will improve.
只要你持續練習，就會進步。

承接 · 順序
Sequence — fixed order, no swapping
→
before · after · once · as soon as
— Example —
He washed his hands before he sat down.
他先洗手，再坐下。

05
因果 · 推動
Cause — one pushes the other
→
because · since · as · so that
— Example —
I'm exhausted today because I didn't sleep well.
因為我昨晚沒睡好，所以今天很累。

06
假設 · 想像
Hypothesis — not yet real
if · suppose · in case
— Example —
If tomorrow is a holiday, I'll go hiking.
如果明天放假，我就去爬山。

轉折 · 反向
Contrast — expected one way, went the other
≠
although · though · even though · while
— Example —
Although it was raining, we still set out.
雖然下雨，我們還是出發了。

Purpose — all roads to one point
↓ ↓ ↓ ↓
so that · in order to · to
— Example —
