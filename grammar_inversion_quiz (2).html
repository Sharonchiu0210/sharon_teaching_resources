假設語氣：若非

If it were not for / If it had not been for / Without / But for

核心觀念

這四個結構都表示「若非……（否則結果會不同）」，是假設語氣（Subjunctive Mood）的一部分。重點：與現在事實相反 vs. 與過去事實相反。

倒裝句裡沒有助動詞 do/does/did，但有 be 動詞或情態助動詞提到前面。

與現在事實相反

結構 If it were not for + N，主句 would / could / might + V
等義替換 Without + N，... But for + N，... Were it not for + N，...（倒裝，省略 if）
If it were not for your help, I couldn't finish the project.
= Without your help, I couldn't finish.
= But for your help, I couldn't finish.
= Were it not for your help, I couldn't finish.
若非你的幫助，我無法完成這個計畫。

與過去事實相反

結構 If it had not been for + N，主句 would / could / might + have + V-pp
等義替換 Without + N，... would have + V-pp But for + N，... would have + V-pp Had it not been for + N，...（倒裝）
If it had not been for the warning sign, I might have gotten hurt.
= Without the warning sign, I might have gotten hurt.
= But for the warning sign, I might have gotten hurt.
= Had it not been for the warning sign, I might have gotten hurt.
若非有警告標誌，我可能受傷了。

四者比較總表

結構時態用法備註
If it were not for與現在相反標準假設語氣
If it had not been for與過去相反標準假設語氣
Without / But for現在／過去皆可看主句助動詞判斷
Were it not for（倒裝）與現在相反省略 if，be 動詞提前
Had it not been for（倒裝）與過去相反省略 if，had 提前
考試考點：without / but for 後面接名詞，主句動詞要用情態助動詞（would/could/might），且根據時態判斷是否加 have + pp。

段考例題解析

(A) But for my morning coffee, I might feel sleepy in class.
✅ 與現在相反，might + V（原形）— 正確
(B) If it were not for my classmates' help, I might fall behind in math.
✅ 與現在相反，might + V — 正確
(D) If it had not been for the traffic jam, we might have arrived on time.
✅ 與過去相反，might have + pp — 正確
(C) Without my sister encouraged me... ❌
❌ Without 後面只能接名詞或名詞片語，不能接子句（encouraged me 是動詞片語）。應改為 Without my sister's encouragement，或 If it were not for my sister's encouragement...

倒裝句

Only / Not until / Not only / Adj. as S V / Barely / 倒裝核心概念

倒裝核心概念

倒裝句 = 把助動詞（或 be 動詞）提到主詞前面。記憶口訣：

「倒裝句就是文具——助動詞提出來。」一般句：S + V；倒裝句：助動詞 + S + V
助動詞種類 be 動詞：am / is / are / was / were 助動詞：do / does / did（用於一般動詞） 情態助動詞：can / will / should / may / could / would 完成式：have / has / had

❗ 最常考：否定副詞放句首 → 後面要倒裝（否定倒裝）

Only + 副詞/子句 放句首

Only + 時間副詞 / 介系詞片語 / when / after / by，助動詞 + S + V
Only when students finish all their homework are they allowed to play outside.
只有當學生完成所有作業時，才能出去玩。
Only by working hard can you achieve your goal.
只有努力工作，你才能達成目標。
注意！Only 修飾的 when/if 子句本身不倒裝，主要子句才倒裝。
❌ Only when are children accompanied... → 錯誤
✅ Only when children are accompanied..., are they allowed... → 正確

Not until 放句首

Not until + 時間點/子句，助動詞 + S + V
Not until she lost her job did Mary understand the value of saving money.
直到她失業，Mary 才明白存錢的重要。
Not until midnight did she finish her report.
直到午夜，她才完成報告。

Not only 放句首

Not only + 助動詞 + S + V，but (S) also + V
Not only did they win the contest, but they also received a cash prize.
他們不只贏得比賽，還拿到了獎金。
Not only is she talented, but she is also hard-working.
她不只有才華，還很努力。

Adj. as S V（讓步倒裝）

Adj. / Adv. / N + as + S + V，主句

意思相當於 Although/Though + S + be + Adj.，但語氣更正式。

Hard as the training may be, every member will gain real experience.
雖然訓練可能很辛苦，每位成員都能獲得真正的經驗。
Young as he is, he has achieved a lot.
雖然他很年輕，卻有很多成就。
Doctor as she is, she cannot cure everything.
雖然她是醫生，也無法治癒所有疾病。
名詞放句首時不加冠詞：Doctor（不是 A doctor）as she is...

Barely / Hardly / Scarcely（否定副詞倒裝）

Barely / Hardly / Scarcely + 助動詞 + S + V，when / before + S + V
Barely do I know what happened.
我幾乎不知道發生了什麼事。
Hardly had she arrived home when the phone rang.
她幾乎剛到家，電話就響了。

Rarely / Seldom（否定副詞倒裝）

Rarely / Seldom + 助動詞 + S + V
Rarely does the girl raise her hand in class since she is afraid of making mistakes.
這個女孩很少在課堂上舉手，因為她害怕犯錯。

關係子句釐清

where / when / in which / which — 用法比較與混淆點整理

關係代名詞 vs. 關係副詞

先確認先行詞的角色：它在子句裡是主詞/受詞（用 which/that）還是地點/時間（用 where/when）？

類型先行詞關係詞子句中角色
關係代名詞事/物which / that主詞或受詞
關係副詞地方where = in which地方副詞
關係副詞時間when = in which / on which時間副詞
關係副詞原因why = for which原因副詞

where vs. in which

兩者完全可以互換，但 in which 更正式。

先行詞（地方）+ where + S + V（子句裡沒有介系詞） 先行詞（地方）+ in which + S + V（in 移到前面了）
This is the city where I grew up.
= This is the city in which I grew up.
這是我成長的城市。
判斷方式：把子句展開看看：I grew up in the city → in 已被吸收進 where 或提到前面變 in which。兩者不能同時存在：❌ the city where I grew up in

when vs. in which / on which

先行詞是時間名詞時，when 可用 on which / in which 取代。

I still remember the day when we first met.
= I still remember the day on which we first met.
我仍然記得我們第一次見面的那天。
The year when she was born = The year in which she was born.
她出生的那一年。

which 的三種用法釐清

1. 關係代名詞（修飾事/物，子句裡作主詞或受詞）

The book which I borrowed is fascinating.
我借的那本書很迷人。（which 作受詞）
The report which was submitted yesterday contained errors.
昨天提交的報告有錯誤。（which 作主詞）

2. 介系詞 + which（= where / when / why）

the reason for which = the reason why
the place in which = the place where
the day on which = the day when

3. 補充說明用的非限定子句

She passed the exam, which surprised everyone.
她通過了考試，這讓大家都很驚訝。（which 指前整句話）

常見混淆點整理

情況正確錯誤
地方先行詞 + 子句有 inwhere I live ✅
in which I live ✅		where I live in ❌
which vs. whothe man who ✅
the cat which ✅		the man which ❌
which vs. that先行詞有最高級/序數 → that
先行詞有 the only → that		the best book which ❌（通常用 that）
補述用法, which ✅, that ❌（補述不用 that）
非限定（補充說明）子句：先行詞後有逗號 → 只能用 which，不能用 that。
